# Генерация видео 
Автоматическое создание видеороликов с помощью Python и популярных библиотек, таких как `requests` и `moviepy`. 
Включает этапы получения данных из внешних источников, их визуализации и последующей компиляции в готовый видеоролик.

## Используемые технологии
- **Python** — основной язык разработки.  
- **requests** — для взаимодействия с внешними API и загрузки данных.  
- **moviepy** — для рендеринга, редактирования и экспорта видео.  
- *(Опционально)* **matplotlib / pandas** — для визуализации числовых данных.

## Потенциальные улучшения
Добавление веб-интерфейса (например, через Streamlit или FastAPI) для настройки параметров генерации без редактирования кода.
Интеграция с облачными хранилищами (Google Drive, AWS S3) для автоматической загрузки исходных данных и публикации готовых видео.
Поддержка аудиодорожек, озвучки через TTS (например, с использованием ElevenLabs или Yandex SpeechKit).
Генерация вертикального видео под форматы Instagram Reels, TikTok и YouTube Shorts.
