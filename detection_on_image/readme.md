# Поиск объектов на изображении с использованием OpenCV

Этот проект демонстрирует, как с помощью Python и библиотеки OpenCV можно выполнять поиск объектов на изображении. В качестве примера реализовано обнаружение блюдца с использованием метода обработки изображений **Canny Edge Detection** — алгоритма для выделения границ объектов.

## Основные возможности

- **Обнаружение объектов**:  
  Использование алгоритма **Canny** для выявления границ объектов на изображении на основе градиентов интенсивности пикселей.

- **Работа с изображениями**:  
  Поддержка загрузки изображений с диска, предварительной обработки (например, преобразование в оттенки серого) и применения детектора границ.

- **Визуализация результатов**:  
  Отображение как исходного изображения, так и результата обработки — для наглядного сравнения и оценки эффективности алгоритма.

## Используемые технологии

- **Python** — основной язык программирования.  
- **OpenCV** — библиотека компьютерного зрения, предоставляющая функции для обработки и анализа изображений.  
- **Matplotlib** — инструмент для визуализации изображений и результатов в удобной графической форме.
